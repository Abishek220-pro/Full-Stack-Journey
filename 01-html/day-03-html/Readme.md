# 📘 HTML5 Day 3 – Multimedia & SEO Metadata

A complete beginner-to-intermediate guide to HTML5 Multimedia and SEO Metadata.

This lesson covers everything you need to know before starting CSS. It explains images, responsive images, audio, video, accessibility, SEO, social media metadata, and includes a practical project.

---

## 🎯 Learning Objectives

After completing this lesson, you will be able to:

- Add and optimize images
- Create responsive images using `<picture>`
- Use `srcset` and `sizes`
- Implement lazy loading
- Embed videos and audio
- Add captions using `<track>`
- Improve accessibility with `alt`
- Understand image formats
- Write complete SEO metadata
- Configure Open Graph tags
- Configure Twitter Cards
- Add favicons
- Build a semantic HTML profile page
